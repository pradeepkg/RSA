MangoApps

September 30, 2023 · 55 min read
Pradeep Kadambar
Creator

MangoApps is a versatile collaboration platform that enables organizations to streamline communication and enhance productivity. It offers a comprehensive suite of features, including instant messaging, file sharing, task management, and team collaboration tools. With its intuitive interface and robust functionality, MangoApps empowers teams to connect, share ideas, and work together seamlessly across departments and geographies.

Versions: 7.5.2

Overview​

In an earlier blog, MangoApps Collections we looked into collecting accounts and entitlements from MangoApps. In this article, we will discuss provisioning into MangoApps using REST APIs.

Connector Settings​

Steps

  1. Login to RSA Identity G&L console as System Administrator.
  2. Navigate to AFX > Connectors.
  3. Click on Create Connector and select the RESTful Web Service connector template.
  4. Enter the details as shown below under General tab.

  5. Click on the Settings tab and enter the required connection details.

  6. Click OK.

Add Account to Group​

Steps

  1. Edit the connector and navigate to Capabilities tab.
  2. Click on Add Account to Group and configure as shown below.
info

The MangoApps APIs to add account to group does not support passing account name (email) and hence we cannot use this in conjunction with Create Account. Create account does not return account ID and hence will have to wait for data collection.

Command Input Parameters

Parameter Name*Display NameMappingDescription
AccountId✔️Account ID${Account.External_Id}Unique identifier for the user
GroupId✔️Group ID${Group.External_Id}Unique identifier for the group

🔒 = Encrypted ✔️ = Required

Command Code
Generate SessionToken✔️
MethodPUT
Pathapi/groups/${GroupId}/members/manage.json?send_member_list=false
Request Headers
Content-Typeapplication/x-www-form-urlencoded
Acceptapplication/json; charset=utf-8
AuthorizationBearer ${SessionToken}
Request Body
{
  "ms_request": {
    "group": {
      "add_member_ids": [
        "${AccountId}"
      ],
      "generate_feed": "Y"
    }
  }
}
