Notion

May 29, 2024 · 25 min read
Pradeep Kadambar
Creator

Notion is a versatile productivity tool offering note-taking, project management, and collaboration features. Its customizable workspace allows users to organize tasks, documents, and databases efficiently. With seamless integration across platforms, Notion enhances team collaboration and individual productivity, making it a popular choice for personal and professional use.

Versions: 8.0.0 P2

Overview​

It is recommended that you review the Notion Collections guide before proceeding with this blog.

Connector Settings​

Steps

  1. Login to RSA Identity G&L console as System Administrator.
  2. Navigate to AFX > Connectors.
  3. Click on Create Connector and select the RESTful Web Service connector template.
  4. Enter the details as shown below under General tab.

  5. Click on the Settings tab and enter the required connection details.

  6. Click on Test Connector Settings to verify the settings.

Supported Verbs​

Create an Account​

Steps

  1. Edit the connector and navigate to Capabilities tab.
  2. Click on Create an Account and configure as shown below.

Command Input Parameters

Parameter NameDisplay NameMappingDescription
UserName✔️User Name${AccountTemplate.Name}The account name
Email✔️Email Address${AccountTemplate.Email}Primary email address of user
FamilyName✔️Last Name${AccountTemplate.LastName}Family name of the user
GivenName✔️First Name${AccountTemplate.FirstName}Given name of the user
FormatedName✔️Formatted Name${AccountTemplate.DisplayName}User's display name
Password✔️ 🔒The account password${AccountTemplate.Password}The initial password for the account

🔒 = Encrypted ✔️ = Required

  1. Click OK.
  2. Run the Test Connector Capabilities to verify the verb.

Delete an Account​

Steps

  1. Edit the connector and navigate to Capabilities tab.
  2. Click on Delete an Account and configure as shown below.

Command Input Parameters

Parameter NameDisplay NameMappingDescription
UserIdExternal ID${Account.External_Id}Unique identifier for the user
UserNameUser Name${AccountTemplate.Name}The account name

🔒 = Encrypted ✔️ = Required

  1. Click OK.
  2. Run the Test Connector Capabilities to verify the verb.

Add Account to Group​

Steps

  1. Edit the connector and navigate to Capabilities tab.
  2. Click on Add Account to Group and configure as shown below.

Command Input Parameters

Parameter NameDisplay NameMappingDescription
UserIdExternal ID${Account.External_Id}Unique identifier for the user
UserNameUser Name${Account.Name}The account name
GroupIdExternal ID${Group.External_Id}Unique identifier for the group
GroupNameGroup Name${Group.Name}The account name

🔒 = Encrypted ✔️ = Required

  1. Click OK.
  2. Run the Test Connector Capabilities to verify the verb.

Steps
