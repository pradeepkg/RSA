Unify

Run a unification on collected identities.

GET https://instance.securid.com:443/aveksa/command.submit?cmd=unify

Request Parameters​

unify
formatproperties - (Default) key=value. most useful when requesting a single object. If multiple objects are returned, values is a csv. ie key=value1,value2,value3 where value1 is the value for the first object, value2 is the value for the 2nd object, etc.

json - Useful format for parsing into javascript or other languages. See www.json.org for details.
The id of the run to delete.

Request Headers​

Bearer <token>
Acceptapplication/json
Content-Typeapplication/json

Response Parameters​

typeExecuted command name
run-idThe id assigned to the unification.

Request Example​

  • Curl
  • Python
curl -K -X GET  \
"https://instance.securid.com:443/aveksa/command.submit?cmd=unify&format=json" \
-H "Authorization: Bearer <token>" \
-H "Content-Type: application/json"
import requests, json

url = "https://instance.securid.com:443/aveksa/command.submit"

params = {
    "cmd": "unify",
    "format": "json"
}

headers = {
    'Authorization': 'Bearer <token>',
    'Content-Type': 'application/json'
}

response = requests.get(url, params=params, headers=headers)

if response.status_code == 200:
    data = response.json()
    print(data)
else:
    print(f"Request failed with status code {response.status_code}")
