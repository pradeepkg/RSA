Close CRM

December 23, 2023 · 22 min read
Pradeep Kadambar
Creator
Close CRM streamlines sales with a user-friendly platform, integrating communication, leads, and tasks. Its robust features enhance collaboration, automate workflows, and drive sales efficiency. With a focus on simplicity and powerful functionality, Close CRM empowers teams to close deals faster and build stronger customer relationships.

Versions: 8.0.0

Identities ❌ | Accounts ✔️ | Groups ❌ | App Roles ❌ | Entitlements ❌

Application​

Steps

  1. Login to RSA G&L console as System Administrator.
  2. Navigate to Resources > Application.
  3. Click on Create Application and select Other Application.
  4. Enter the details as shown below.
  5. Click Finish.

Account Collector​

This section explains the process for configuring an account collector for the Close CRM application. We will use the RSA G&L Generic REST collector for this purpose.

Steps

  1. Login to RSA G&L console as System Administrator.
  2. Navigate to Resources > Applications.
  3. Click on Close CRM.
  4. Navigate to Collectors tab and click on Create Account Collector. Enter the details as shown below.

  5. Click Next and enter the connection details. Since we are using Token Access method for authentication, the collector will use Basic authentication type.

  6. Click Next. Select the data types that the collector will be collecting from Close CRM

  7. Click Next. On the Account Data page, enter the following configurations.



Request
MethodGET
URL/api/v1/user/
Query Parameters
_fieldsid,email
Request Headers
Acceptapplication/json
Response Mappings
AccountId$.data.*.email
Id$.data.*.id
  1. Click Next. On the Group Data page, enter the following configurations.



note

This API does not support pagination

Request
MethodGET
URL/api/v1/group/
Request Headers
Acceptapplication/json
Response Mappings
GroupId$.data.*.name
Id$.data.*.id
  1. Click Next. On the Account Group Membership page, enter the following configurations.




Request
MethodGET
URL/api/v1/group/${Id}
Query Parameters
_fieldsid,members
Request Headers
Acceptapplication/json
Response Mappings
Account Identifier$.members.*.user_id
  1. Click Next. In the Map Collector Attributes to Account Attributes, map the collected account attributes.

  2. Click Next. In the Map Collector Attributes to Account Mapping Attributes, map the user reference attribute.

  3. Click Next. In the Map Collector Attributes to Group Attributes, map the collected group attributes.

  4. Click Next. In the User Resolution Rules, map the user to account.

  5. Click Next. In the Member Account Resolution Rules, map the account attribute.


    💡 Info: This value MUST always be mapped to the Account Name for any generic REST collector.
  6. Click Finish. Use the Test function to make sure the configurations are accurate.

Application Setup​

This section explains the process of setting up Close CRM for integrating with G&L.

Steps

  1. Login to Close CRM dashboard and navigate to settings https://app.close.com/settings/
  2. Under Connect, click on Developer.
  3. Click on + NewAPI Key.

  4. Provide a suitable name for the key and click Create API Key.

    .
  5. Copy and save the API for use with the collectors.
