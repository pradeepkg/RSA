BambooHR

November 25, 2023 · 19 min read
Pradeep Kadambar
Creator
BambooHR is a cloud-based human resources software that streamlines employee management. It offers features for HR processes, including employee data management, time-off tracking, and performance management. With user-friendly interfaces, it simplifies HR tasks, enhances employee engagement, and provides insights to drive better decision-making for organizations of all sizes.

Versions: 8.0.0 P1

Identities ✔️ | Accounts ❌ | Groups ❌ | App Roles ❌ | Entitlements ❌

Application​

Steps

  1. Login to RSA G&L console as System Administrator.
  2. Navigate to Resources > Application.
  3. Click on Create Application and select Other Application.
  4. Enter the details as shown below.
  5. Click Finish.

Identity Collector​

This section explains the process for configuring an identity collector for the BambooHR application. We will use the RSA G&L Generic REST collector for this purpose.

Steps

  1. Login to RSA G&L console as System Administrator.
  2. Navigate to Resources > Applications.
  3. Click on BambooHR.
  4. Navigate to Collectors tab and click on Create Identity Collector. Enter the details as shown below.

  5. Click Next and enter the connection details. BambooHR requires API key to be sent over using Basic authentication method. For more information refer to Authentication section.

Application Setup​

This section explains the process of setting up a service account in BambooHR for use within the collections process.

Steps

  1. Login to BambooHR dashboard https://instance.bamboohr.com/home
  2. Click on the profile on the right corner and select API Keys.

  3. Click on Add New Key

  4. Provide a suitable name for the key and click Generate Key

  5. Copy the token for use with G&L.

  6. Click Done.
