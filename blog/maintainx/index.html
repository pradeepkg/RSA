MaintainX

June 10, 2024 · 28 min read
Pradeep Kadambar
Creator
MaintainX is a versatile maintenance management platform designed to streamline operations for businesses across various industries. With intuitive features for work orders, asset management, and team collaboration, MaintainX empowers organizations to efficiently track and manage maintenance tasks, enhance productivity, and ensure equipment reliability. Its user-friendly interface and mobile accessibility make it a convenient solution for modern maintenance needs.

Versions: 8.0.0 P2

Identities ❌ | Accounts ✔️ | Groups ✔️ | App Roles ❌ | Entitlements ❌

Application​

Steps

  1. Login to RSA G&L console as System Administrator.
  2. Navigate to Resources > Application.
  3. Click on Create Application and select Other Application.
  4. Enter the details as shown below.
  5. Click Finish.

Account Collector​

This section explains the process for configuring an account collector for the MaintainX application. We will use the RSA G&L Generic REST collector 📘 for this purpose.

Steps

  1. Login to RSA G&L console as System Administrator.
  2. Navigate to Resources > Applications.
  3. Click on MaintainX.
  4. Navigate to Collectors tab and click on Create Account Collector. Enter the details as shown below.

  5. Click Next and enter the connection details.

  6. Click Next. Select the data types that the collector will be collecting from MaintainX.

  7. Click Next. On the Account Data page, enter the following configurations.

Account Identifier$.members.*.id
  1. Click Next. In the Map Collector Attributes to Account Attributes, map the collected account attributes.

  2. Click Next. In the Map Collector Attributes to Account Mapping Attributes, map the user reference attribute.

  3. Click Next. In the Map Collector Attributes to Group Attributes, map the collected group attributes.

  4. Click Next. In the User Resolution Rules, map the user to account.

  5. Click Next. In the Member Account Resolution Rules, map the account.

  6. Click Finish. Use the Test function to make sure the configurations are accurate.

Application Setup​

This section explains the process of setting up MaintainX for integration with G&L.

Steps

  1. Login to MaintainX dashboard https://app.getmaintainx.com/.
  2. On the bottom left hand corner, click on the user profile and click on Integrations.

  3. Click on New Key.

  4. Provide a suitable name for the integration and click on Generate Key.

  5. Copy the token and retain for use with G&L.
