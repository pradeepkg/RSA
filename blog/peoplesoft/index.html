Oracle PeopleTools (PeopleSoft)

October 2, 2023 · 23 min read
Pradeep Kadambar
Creator
Oracle PeopleTools is a comprehensive suite of software development tools that facilitates the creation, customization, and maintenance of applications within the Oracle PeopleSoft enterprise software ecosystem. It includes tools for development, integration, reporting, and workflow management, enhancing efficiency and user experience in business processes.

Application​

Steps

  1. Login to RSA G&L console as System Administrator.
  2. Navigate to Resources > Application.
  3. Click on Create Application and select Other Application.
  4. Enter the details as shown below.
  5. Click Finish.

Account Collector​

This section explains the process for configuring an account collector for the Oracle PeopleTools (PeopleSoft) application. We will use the SQL Server database collector for this purpose.

Steps

  1. Login to RSA G&L console as System Administrator.
  2. Navigate to Resources > Applications.
  3. Click on PeopleSoft.
  4. Navigate to Collectors tab and click on Create Account Collector. Enter the details as shown below.

  5. Click Next and enter the Oracle database details. Use the Test Connection to make sure the connection details are accurate.

Entitlement Collector​

This section explains the process for configuring an entitlement collector to collect admin roles from Oracle PeopleTools (PeopleSoft) application. We will use the SQL Server database collector for this purpose.

Steps

  1. Login to RSA G&L console as AveksaAdmin.
  2. Navigate to Resources > Applications.
  3. Click on PeopleSoft.
  4. Navigate to Collectors tab and click on Create Entitlement Collector. Enter the details as shown below.

  5. Click Next and enter the database connection details as shown in Step 5 of Account Collector.
  6. Click Next. Select the data types that the collector will be collecting from PeopleSoft.

  7. Click Next. On the General Column Names, map the attribute names.

  8. Click Next. Under Application Role Data section, use the following query to collect admin roles from PeopleSoft.
SELECT 
  DISTINCT ROLENAME AS role, 
  DESCR AS description 
FROM 
  dbo.PSROLEDEFN‍‍‍‍‍‍‍‍‍‍‍‍‍‍‍‍‍‍‍‍‍‍‍‍‍‍‍‍‍‍‍‍‍‍‍‍‍‍‍‍‍‍‍‍‍‍‍‍‍‍‍‍‍‍‍‍‍‍‍‍‍‍‍‍‍‍‍‍‍‍‍‍‍‍‍‍‍‍‍‍‍‍‍‍‍‍‍‍‍‍‍‍‍‍‍‍‍‍‍‍‍‍‍‍‍‍‍‍‍‍‍‍‍‍‍‍‍‍‍‍‍‍‍‍‍‍‍‍‍‍‍‍‍

The following are the important attributes

  1. Click Next. On the Resource-Action Entitlements Data section, enter the query below to collect resource action mappings for roles.
SELECT 
  DISTINCT ROLENAME AS role, 
  CLASSID AS resource, 
  '' AS action 
FROM 
  dbo.PSROLECLASS 
union 
SELECT 
  DISTINCT '' AS role, 
  CLASSID AS resource, 
  '' AS action 
FROM 
  dbo.PSCLASSDEFN 
WHERE 
  CLASSID NOT IN(
    SELECT 
      DISTINCT CLASSID 
    FROM 
      dbo.PSROLECLASS
  )
