Zendesk

September 16, 2023 · 24 min read
Pradeep Kadambar
Creator
Zendesk is a customer support platform that streamlines interactions between businesses and their customers. It offers ticket management, knowledge base, live chat, and more, empowering organizations to deliver exceptional support experiences. With user-friendly features, Zendesk enhances customer communication and issue resolution, ultimately fostering strong relationships and improving customer satisfaction.

Application​

Steps

  1. Login to RSA G&L console as System Administrator.
  2. Navigate to Resources > Application.
  3. Click on Create Application and select Other Application.
  4. Enter the details as shown below.
  5. Click Finish.

Account Collector​

This section explains the process for configuring an account collector for the Zendesk application. We will use the RSA G&L Generic REST collector for this purpose.

Steps

  1. Login to RSA G&L console as System Administrator.
  2. Navigate to Resources > Applications.
  3. Click on Zendesk.
  4. Navigate to Collectors tab and click on Create Account Collector. Enter the details as shown below.

  5. Click Next and enter the connection details.Since we are using Token Access method for authentication, the collector will use Basic authentication type.

info

The service account email address needs to be prefixed with /token and password will be the API token.

Application Setup​

This section explains the process of setting up a service account in Zendesk for use within the collections process.

Steps

  1. Login to Zendesk admin dashboard Apps and integrations > APIs > Zendesk API.
  2. If you are accessing for the first time, agree to terms and conditions.
  3. Enable the Token Access.

  4. Click on Add API token.

  5. Provide a suitable name and copy the token for use with G&L.
  6. Click Save.
