TalentLMS

April 24, 2024 · 30 min read
Pradeep Kadambar
Creator
TalentLMS is a user-friendly learning management system designed for businesses and educational institutions. With intuitive course creation tools, customizable learning paths, and robust tracking features, it enables organizations to deliver effective training to employees, customers, and partners. Its cloud-based platform offers scalability, ease of use, and comprehensive reporting, making it an ideal solution for modern training needs.

Versions: 8.0.0 P2

Identities ❌ | Accounts ✔️ | Groups ✔️ | App Roles ❌ | Entitlements ❌

Application​

Steps

  1. Login to RSA G&L console as System Administrator.
  2. Navigate to Resources > Application.
  3. Click on Create Application and select Other Application.
  4. Enter the details as shown below.
  5. Click Finish.

Account Collector​

This section explains the process for configuring an account collector for the TalentLMS application. We will use the RSA G&L Generic REST collector 📘 for this purpose.

Steps

  1. Login to RSA G&L console as System Administrator.
  2. Navigate to Resources > Applications.
  3. Click on TalentLMS.
  4. Navigate to Collectors tab and click on Create Account Collector. Enter the details as shown below.

  5. Click Next and enter the connection details.



  6. Click Next. Select the data types that the collector will be collecting from TalentLMS

  7. Click Next. On the Account Data page, enter the following configurations.



info

This is the only API in TalentLMS that supports pagination

Request
MethodGET
URL/api/v1/users (Sample Response)
Request Headers
Acceptapplication/json
Response Mappings
AccountId$.*.login
Id$.*.id
Email$.*.email
Status$.*.status
  1. Click Next. On the Group Data page, enter the following configurations.


Request
MethodGET
URL/api/v1/groups (Sample Response)
Request Headers
Acceptapplication/json
Response Mappings
GroupId$.*.name
Id$.*.id
Description$.*.description
Key$.*.key
Owner$.*.owner_id
  1. Click Next. On the Account Group Membership page, enter the following configurations.


Request
MethodGET
URL/api/v1/groups/id:${Id} (Sample Response)
Request Headers
Acceptapplication/json
Response Mappings
Account Identifier$.users.*.id
  1. Click Next. In the Map Collector Attributes to Account Attributes, map the collected account attributes.

  2. Click Next. In the Map Collector Attributes to Account Mapping Attributes, map the user reference attribute.

  3. Click Next. In the Map Collector Attributes to Group Attributes, map the collected group attributes.

  4. Click Next. In the User Resolution Rules, map the user to account.

  5. Click Next. In the Member Account Resolution Rules, map the account.

💡 Info: This value MUST always be mapped to the Account Name for any generic REST collector.

  1. Click Finish. Use the Test function to make sure the configurations are accurate.

Application Setup​

This section explains the process of setting up TalentLMS for integration with G&L.

Steps

  1. Login to TalentLMS https://instance.talentlms.com/
  2. Click on Go To and select Account & Settings.

  3. Under Security check the Enable API.

  4. Copy the token and retain for use with G&L.
